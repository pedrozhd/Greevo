Bem-vindo ao nosso FAQ!

Como o Greevo funciona?

O Greevo é uma plataforma que conecta pessoas em risco a abrigos seguros durante desastres naturais. Basta informar sua localização (ou usar o GPS) para ver os abrigos mais próximos em tempo real, com filtros por acessibilidade e lotação.

O Greevo funciona sem internet?

Sim! Nossa tecnologia offline-first armazena dados localmente. Se a internet cair:

  • Você ainda vê abrigos pré-carregados.
  • Voluntários podem atualizar lotações via SMS.

Como os abrigos são verificados?

Todos os abrigos são cadastrados e validados por Defesas Civis ou parceiros oficiais, garantindo informações confiáveis. Eles aparecem com um selo de verificação (✔️) no mapa.

Turistas podem usar o Greevo?

Com certeza! O Greevo oferece:

  • Alertas traduzidos automaticamente (inglês/espanhol/francês).
  • Pontos de encontro internacionais em zonas turísticas.

Como ajudar se sou um voluntário?

  • Cadastre-se como voluntário na Defesa Civil local.
  • Greevo para atualizar lotações em abrigos
  • Greevo para reportar necessidades

O Greevo consome muita bateria?

Não! O app é otimizado para:

  • GPS de baixo consumo.
  • Modo noturno para economizar energia.

É gratuito?

Sim! O Greevo é 100% gratuito para usuários. Mantemos o projeto via parcerias com governos e doações.

Como denunciar um abrigo irregular?

Use o botão "Reportar Problema" no perfil do abrigo ou contate a Defesa Civil diretamente pelo chatbot.

Posso usar o Greevo para treinamentos?

Sim! Oferecemos simulados de evacuação para comunidades. Entre em contato via:

  • E-mail: treinamentos@greevo.org
  • Chatbot: Digite "SIMULADO".

Quais tecnologias o Greevo usa?

  • Backend: Java (console administrativo).
  • Frontend: HTML/CSS/JS puro (sem frameworks).
  • Banco de dados: Armazenamento local durante a sessão.
